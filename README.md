## Methodology & Roadmap

Below is the detailed structure of the knowledge base. Use this as a roadmap during your labs or exams.

```text
├── 01. OSINT (Domain Info, Cloud, Google Dorks, Web Archives)
├── 02. Network Enumeration
│   ├── Live Host Discovery (ARP, FPing, Bash)
│   ├── Port Scanning (Nmap, Rustscan, Masscan)
│   ├── Traffic Analysis (TCPDump)
│   └── Protocol Specific (FTP, SSH, SMB, SQL, RDP, WinRM, etc.)
├── 03. Vulnerability Assessment (Nessus, Nikto, Nuclei, OpenVAS)
├── 04. File Transfers (Linux/Windows, LOLBins, Evasion, Exfiltration)
├── 05. Shells & Payloads (MSFvenom, Reverse Shells, Web Shells, Spawning)
├── 06. Credential Access
│   ├── Storage & Hunting (Linux/Windows Creds)
│   ├── Cracking (Hashcat, John The Ripper)
│   ├── Attacks (LSASS, SAM, Pass-the-Hash/Ticket)
│   └── Defense & Management (MFA, Policies)
├── 07. Pivoting & Tunneling (Chisel, SSH, Socat, SOCKS, DNS/ICMP Tunnels)
├── 08. Active Directory
│   ├── Enumeration (BloodHound, PowerView, Living off the Land)
│   └── Attacks (Kerberoasting, AS-REP, DCSync, ACL Abuse, Golden/Silver Tickets)
├── 09. Web Exploitation
│   ├── Tools & Proxies (Burp Suite, ZAP, Ffuf)
│   ├── Injection Attacks (SQLi, Command Injection)

    ├── Linux (Kernel, Sudo, SUID)
    └── Windows (Token Impersonation, Potato Attacks, Enumeration)
```
